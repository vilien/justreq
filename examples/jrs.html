怎么做到的呢？

你只要在.justreq文件添加如下规则，并编写相应的users.jrs、save-user.jrs文件即可

{
  ...
  "rules": [
    {
      "href" : "api/users.do",
      "subs" : "jrs/users.jrs"
    },
    {
      "href" : "api/save-user.do",
      "subs" : "jrs/save-user.jrs"
    }
  ]
}

本示例所使用jrs文件位于“.jr/subs/jrs/”
